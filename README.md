
 Web DUO Pin multiplexing contigurator   

Build on development board:

```
git clone https://github.com/protektwar/duo_web_pinmux
cd duo_web_pinmux
make
```

Run: 
```
./duo_web_pinmux_server
```

Then open Browser on address of the development board port 8080
